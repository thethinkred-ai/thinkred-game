# 🤝 Как помочь ThinkRed

Спасибо, что хотите помочь проекту! Мы рады любому вкладу.

## 🐛 Нашли баг?

1. Проверьте, нет ли уже такого [Issue](https://github.com/thethinkred-ai/thinkred-game/issues)
2. Если нет — создайте новый с описанием:
   - Что делали
   - Что ожидали увидеть
   - Что увидели на самом деле
   - Скриншот (если можно)

## 💡 Есть идея?

Создайте Issue с пометкой `enhancement` и опишите:
- Какая проблема решается
- Как вы видите решение
- Примеры похожих решений (если есть)

## 🛠 Хотите написать код?

### Первый раз на GitHub?

Пошаговая инструкция в README — раздел **«Как помочь проекту»**.

### Быстрый чеклист перед Pull Request

- [ ] Код работает (`npm run dev` не падает)
- [ ] Нет ошибок линтера (`npm run lint`)
- [ ] Название коммита понятное: `Add user avatar upload` → хорошо; `fix` → плохо
- [ ] Описали в PR, что сделали и зачем

### Стиль кода

- Типизируем всё (TypeScript)
- Комментарии на русском или английском — без разницы
- Функции называем глаголами: `getUser()`, `calculateProfit()`
- Компоненты называем существительными: `EnterpriseCard`, `GameBoard`
